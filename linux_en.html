How to play GTanks on Linux (example MX Linux (Debian similar distribution))

Since most users use Windows OS, there was not much demand to port the client to another platform. This tutorial is an installation (loader of executable files with the .exe extension) or wine.
  1. You need to download two executable files .exe flashplayer_32.exe и GlobalSettings.exe;
  2. Open the terminal CTRL + ALT + T;
  3. Update the repositories (sudo apt update);
  4. Install wine (.exe file loader) (sudo apt install wine -y);
  5. Install wine32 (add-on) (sudo apt install wine32 -y);
  6. After installation, open the wine configurator (winecfg);
  7. Select "Windows Version: Windows 10" and click "OK";
  8. Download the client. The downside of this client is that you will have to manually update it (that is, you need to download the archive again when a post about innovations comes out, so watch out!!!);
  9. After that, open “GlobalSettings.exe” (wine GlobalSettings.exe);
  10. Select the "Advanced" tab;
  11. Click on “Setting up a trusted folder...”;
  12. Click on "Add...", "Add folder..." and select the path to the "client" folder IMPORTANT! the path will be displayed in wine like this: "Z:\home\sandbox\Desktop\client" in Linux it is "/home/sandbox/Desktop/client" this is normal, because We emulate the Windows file system in Linux;
  13. Check that the path to the folder has been added and then click on “Close”;
  14. Open "flashplayer_32.exe" (wine flashplayer_32.exe);
  15. Click "File" -> "Open" -> "Browse" and select "GTloader.swf";
  16. Click "OK";
  17. Ready!
I hope this tutorial helped you install the client on your Linux. Good luck on the battlefields!
