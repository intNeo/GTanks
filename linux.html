Как поиграть в GTanks на Linux (пример MX Linux (Debian подобный дистрибутив))

Так как большинство пользователей, используют OS Windows, то не было большого спроса портировать клиент на другую платформу. Данный туториал представляет из себя установку (загрузчика исполняемых файлов расширением .exe) или же wine.
  1. Необходимо скачать два исполняемых файла .exe flashplayer_32.exe и GlobalSettings.exe;
  2. Открываем терминал CTRL + ALT + T;
  3. Обновляем репозитории (sudo apt update);
  4. Устанавливаем wine (загрузчик .exe файлов) (sudo apt install wine -y);
  5. Устанавливаем wine32 (дополнение) (sudo apt install wine32 -y);
  6. После установки открываем конфигуратор wine (winecfg);
  7. Выбираем "Версию Windows: Windows 10" и нажимаем "ОК";
  8. Скачиваем клиент. Минус этого клиента вам придется вручную обновлять его (т.е. вам нужно заного скачивать архив когда выходит пост о нововедениях, поэтому следите!!!);
  9. После этого открываем "GlobalSettings.exe" (wine GlobalSettings.exe);
  10. Выбираем вкладку "Дополнительно";
  11. Нажимаем на "Настройка доверенной папки...";
  12. Нажимаем на "Добавить...", "Добавить папку ..." и выбираем путь к папки "client" ВАЖНО! путь будет в wine отображаться так "Z:\home\sandbox\Рабочий стол\client" в linux он следующий "/home/sandbox/Рабочий стол/client" это нормально, т.к. у нас эмуляция файловой системы Windows в Linux;
  13. Проверьте, что добавился путь к папки и после этого нажмите на "Закрыть";
  14. Открываем "flashplayer_32.exe" (wine flashplayer_32.exe);
  15. Нажимаем "Файл" -> "Открыть" -> "Обзор" и выбираем "GTloader.swf";
  16. Нажимаем "ОК";
  17. Готово!
Надеюсь данный туториал тебе помог установить клиент на твой Linux. Удачи на полях сражений!
